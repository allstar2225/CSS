CSS Tag 총정리

1. about CSS tag
(CSS의 기본 문법 및 선택자)

2. relation tag
(스타일시트의 적용방법)

3. color tag
(글자색상 적용하는 방식)

4. background tag
(배경색 또는 이미지 적용하는 방식)

5. text tag
(글자에 여러가지 스타일 적용하는 방식)

6. font tag
(글꼴의 여러가지 스타일 적용)

7. link tag
(하이퍼링크에 여러가지 속성 적용)

8. List tag
(ul, li, ol 등등 list 속성)

9. table tag
(게시판을 만들때 쓰는 table 속성)

10. imageSprite tag
(하나의 이미지를 여러가지로 뿌릴때)

11. Size tag
(각종 크기 속성)

12. padding tag
(padding의 각종 속성에 대해)

13. border tag
(테두리에 대한 속성 설정)

14. margin tag
(margin의 각종 속성에 대해)

15. display tag
(block, inline, inline-block의 각종 속성에 대해)

16. position tag
(static, relration, fixed, absolute, z-index 대한 각종 속성에 대해)

17. float tag
(수평정렬에 대한 각종 속성에 대해)

18. layout align tag
(margin, float, position으로 layout 만들기)

19. selector tag
(선택자에 종류)

20. Combination Selector tag
(결합선택자 (자손, 자식의 활용법))

21. sibling selector tag
(인접 선택자 사용법)

22. dynamic pseudo-class tag
(동적 의사 클래스 [link, visitor, hover, focus, active])

23. UI element states pseudo-class tag
(현재 상태를 각각 별도로 설정 하는 방법)

19. structural psedo-class tag
(구조 의사 클래스 종류)
