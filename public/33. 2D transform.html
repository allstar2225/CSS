- rotate() 메소드 -

+ 해당 태그를 주어진 각도만큼 시계 혹은 반시계로 돌린다

- scale() 메소드 -

+ 해당 태그를 주어진 값만큼 배율을 늘리거나 줄인다

기본 box
scale() 메소드가 적용된 box ( 너비2배, 높이2배 )

- skew() 메소드 -

+ 해당 태그를 주어진 각도만큼 x, y축 방향으로 기울인다. skew-x 또는 skew-y로 하나의 축만 기울일수도 있다.

기본 box
skew() 메소드가 적용된 box ( x:20deg, y:30deg )

- matrix() 메소드 -

+ 위의 모든 메소드들을 한줄로 요약할수 있다. 순서는 matrix(scaleX(), skewX(), skewY(), scaleY(), translateX(), translateY())

matrix() 메소드로 적용한 박스
