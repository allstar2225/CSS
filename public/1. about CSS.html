- css 기초 -

        + CSS의 기초적인 문법은 선택자(selector) 와 선언부 (declaration) 로 구성된다.
        ex ) 선택자(HTML요소) { 속성명 : 속성값; 속성명 : 속성값 }
        ex2 ) a { font-size : 1rem }

- css 의 선택자 -

        + CSS의 선택자에는 아래와 같은 종류가 있다
        1) HTML 태그 선택자
            # a, div, p 등등 기본 지원 태그들
        2) ID 선택자
            #사용자지정id { 속성명 : 속성값 }
            #  <div id="사용자지정id"> 내용 <div>
        3) CLASS 선택자
            .사용자지정class { 속성명 : 속성값 }
            #  <div class="사용자지정class"> 내용 <div>
        4) group 선택자( html태그, id, class 모두 포함)
            ul, li {
            속성명 : 속성값
        }
