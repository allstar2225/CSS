- static(정적 위치) -

+ html의 가장 기본적인 위치(좌측상단 0, 0)에서 어떠한 값도 영향을 받지 않는다.

static 포지션

- relative(상대 위치) -

+ html의 가장 기본적인 위치(좌측상단 0, 0)를 기준으로 위치를 지정

relative 포지션

- fixed(고정 위치) -

+ viewport를 기준으로 스크롤이 되어도 정해진 위치에 고정되어있는다.

fixed 포지션

- absolute(절대 위치) -

+ viewport를 기준으로 지정되지만, 부모포지션에서 지정된 position 속성에 따라서 결정된다.

absolute 포지션

- z-index -

+ 요소를 겹치게 둘수 있다. -9999 ~ 9999까지 가능하며 클수록 맨앞에 나오게 된다.

z-index 1 라인
static 라인 (이 뒤의 박스가 z-index -1로 설정되어있음)
