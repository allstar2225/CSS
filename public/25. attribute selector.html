- 속성 이름 선택자 -

+ 특정 속성을 가진 태그 모두 선택

[come]으로 속성명 설정

come="아무런말"

아무것도 없음

- 속성 이름 = 속성값 선택자 -

+ 속성값에 설정과 같은 문자열이 들어간 태그에 적용

속성값 = first come

속성이름 = first come

속성이름= first

아무것도 없음

- 속성이름 ~= 속성값 선택자 -

+ 속성값에 설정된 특정문자열이 들어간 태그에 적용(띄어쓰기를 기준으로 단어를 인식)

속성값 = first

속성이름 ~= first 아무말

속성이름 ~= first-아무말

속성이름만 있음

아무것도 없음

- 속성이름 |= 속성값 선택자 -

+ 속성값에 설정된 특정문자열이 들어간 태그에 적용(하이픈(-)을 기준으로 단어 인식

속성값 = first

속성이름 |= first 아무말

속성이름 |= first-아무말

속성이름만 있음

아무것도 없음

- 속성이름 ^= 속성값 선택자 -

+ 속성값에 설정된 문자열로 시작하면 무조건 적용.

속성값 = first

속성이름 ^= first 아무말

속성이름 ^= first-아무말

속성이름 ^= p first

아무것도 없음

- 속성이름 $= 속성값 선택자 -

+ 속성값에 설정된 문자열로 끝나는 태그에 적용

속성값 = first

속성이름 $= first 아무말

속성이름 $= first-아무말

속성이름 $= 아무말 first

속성이름 $= 아무말-first

속성이름 $= p first

아무것도 없음

- 속성이름 *= 속성값 선택자 -

+ 속성값에 설정된 문자열이 들어가기만 한다면 무조건 적용

속성값 = first

속성이름 *= first 아무말

속성이름 *= first-아무말

속성이름 *= 아무말 first

속성이름 *= 아무말-first

속성이름 *= p first

아무것도 없음
