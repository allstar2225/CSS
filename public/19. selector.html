- 전체 선택자 -

+ 모든 태그를 선택하여 한번에 바꾼다

        * { 속성명 : 속성값 }

- HTML태그 선택자 -

+ html에서 제공하는 태그를 선택하여 설정한다

        p { 속성명 : 속성값 }
        <p> 여기에 적용된다 </p>

- id 선택자 -

+ 직접 지정한 id명으로 원하는 태그에 스타일을 넣는다.

        #id명 { 속성명 : 속성값 }
        <p id="id명"> 여기에 적용된다 </p>
        id선택자는 한문서에 한번밖에 사용불가하며, class선택자보다 우선순위로 적용된다

- class 선택자 -

+ 직접 지정한 class명으로 원하는 태그에 스타일을 넣는다.

        .class명 { 속성명 : 속성값 }
        <p class="class명"> 여기에 적용된다 </p>
        <p class="class명"> 여기에도 적용된다 </p>
        <p class="class명"> 여기에도 적용된다 </p>
        id선택자는 한문서에 한번밖에 사용불가하며, class선택자보다 우선순위로 적용된다.class선택자는 복수적으로 사용가능.

- group 선택자 -

+ 여러가지 html태그, id, class 선택자를 묶어서 한번에 바꾼다.

        // html태그의 경우
        h1, h2, h3 {
            속성명 : 속성값
        }
        <h1> h1 적용된다 </h1>
        <h2> h2 적용된다 </h2>
        <h2> h3 적용된다 </h3>

        // id 경우
        #id명, #id1명, #id2명 { 
            속성명 : 속성값 
        }
        <p id="id명"> 여기에 적용된다 </p>
        <p id="id1명"> 여기에 적용된다 </p>
        <p id="id2명"> 여기에 적용된다 </p>

        // class의 경우
        .class명, .class1명, .class2명 { 
            속성명 : 속성값 
        }
        <p class="class명"> 여기에 적용된다 </p>
        <p class="class1명"> 여기에도 적용된다 </p>
        <p class="class2명"> 여기에도 적용된다 </p>
