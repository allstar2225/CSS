- @keyframes (좌측에서 우측으로 이동 효과) -

+ 특정 규칙을 만들어 두고 해당요소의 스타일이 특정 시간까지 서서히 변화하는 과정

오른쪽에서 왼쪽으로 이동 (1초)

- @keyframes (테두리 색깔 변경) -

+ 특정 규칙을 만들어 두고 해당요소의 테두리가 색깔별로 변경됨

테두리 색깔 변경

- animation-delay (딜레이) -

+ 특정 규칙을 만들어 두고 해당요소의 효과가 나오기까지 시간을 지연시킨다

2초뒤 실행된다

- animation-iteration-count (구간반복) -

+ 특정 규칙을 만들어 두고 해당요소의 효과가 얼마만큼 반복하는지 정한다

2번 반복한다
계속 반복한다

- animation-direction (진행 방향) -

+ 특정 규칙을 만들어 두고 해당요소의 진행방향을 정한다 (설정없음, revers, alternate)

정방향
역방향
정방향 갔다가 역방향

- animation-timing-function (시간당 속도) -

+ 해당요소의 애니메이션 효과 시간당 속도를 설정

linear
ease
ease-in
ease-out
ease-in-out
