- block 태그 -

+ div, h1~6, p, ul, ol, form 등이 있다.

- inline 태그 -

+ span, a, img 등이 있다.

- display 설정값 변경 -

+ block 요소를 inline으로, inline을 block으로 바꿀수 있다.

  • padding
  • css에 대해서
  • 스타일시트 연결

- visibility-

+ HTML 태그가 표현될지 아닐지만 결정.

visible : 나타낸다

hidden : 숨겨둔다

collapse : 동적인 테이블에서만 사용. 테두리를 한줄만 보여준다

- opacity -

+ 이미지를 투명화함. 0.0~1.0까지 이며 0에 가까울수록 투명하다. 익스플로러 8이하 버전은 filter를 사용하여 투명도를 조절
